# Mini Projects
Fun side quests to grow in experience and skill.

## Installation
Vite + React 

Go to desired folder location
Open terminal and run the following

```bash
npm create vite@latest
```
enter a project name and press enter (ex. my-react-app)

Select React for framework

Then select language, either TypeScript or Javascript. Hit enter (note: I will be using Javascript)

SHORT CUT
```bash
npm create vite@latest my-react-app -- --template react
```

navigate to directory
```bash
cd my-react-app
```

then install dependencies in the package.json file
```bash
 npm install
 ```

start dev 
```bash
npm run dev
```

terminal will display a local URL

If you want to quit the npm run dev process
In the same terminal you'll need to press Ctrl + C (on Windows/Linux) or Cmd + C / Ctrl + C (on macOS)
